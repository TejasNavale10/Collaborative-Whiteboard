## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/TejasNavale10/Collaborative-Whiteboard.git
   cd Collaborative-Whiteboard
   ```

2. **Set up the server**
   ```bash
   cd server
   npm install
   # Create a .env file and add your MongoDB URI:
   # MONGODB_URI=your_mongodb_uri
   npm start
   ```

3. **Set up the client**
   ```bash
   cd client
   npm install
   npm start
   ```
